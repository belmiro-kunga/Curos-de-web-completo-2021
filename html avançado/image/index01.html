Imagens de fundo HTML 5

Uma imagem de fundo para um elemento

Você pode especificar imagens
de fundo para qualquer elemento HTML visível.
Neste exemplo, a
imagem de fundo é especificada para um elemento p.
Por padrão, a
imagem de fundo se repetirá
na direção(s)
onde é menor do que o elemento onde ela é especificada. (Tente redimensionar
a janela do
navegador para ver como a imagem de fundo se comporta.
