O elemento de imagem

Redimensione o navegador para ver diferentes versões da imagem sendo carregada em diferentes tamanhos de visor. O navegador procura o primeiro elemento de origem onde a consulta de mídia corresponde à largura da janela de visualização atual do usuário e busca a imagem especificada no atributo srcset.

O elemento img é necessário como a última tag filha do bloco de declaração de imagem. O elemento img é usado para fornecer compatibilidade com versões anteriores para navegadores que não suportam o elemento de imagem, ou se nenhuma das tags de origem corresponder.

Nota: O elemento de imagem não é compatível com IE12 e anteriores ou Safari 9.0 e anteriores.
